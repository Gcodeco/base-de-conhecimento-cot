    Erros e Soluções de Acesso ao Micro
    Senha ou Usuário inválido

    Descrição: O usuário não consegue acessar o micro devido a informações de login inválidas.

    Solução:

    • Passo 1: Abrir um chamado para resetar a senha.
    • Fila: COT
    • Nota: Existe uma automação que reseta a senha.
    • Link para Chamado: Reset de Senha
    Falha de Confiança na Estação com o Controlador de Domínio

    Descrição: A máquina perdeu a configuração com o domínio LASA, resultando em falha de confiança.

  • Passo 1: Orientar a loja a abrir um chamado para a Delfia.
  • Passo 2: Solicitar a recolocação da máquina no domínio LASA.

    • Link para Chamado: Recolocação no Domínio LASA

    Falha ao Carregar o Perfil

    Descrição: Ocorre raramente e é diferente do erro acima. Esse problema acontece quando o perfil do usuário na máquina local fica corrompido.

  • Passo 1: Orientar a loja a abrir um chamado para a Delfia.
  • Passo 2: Solicitar a verificação do problema no perfil do usuário.

    • Link para Chamado: Verificação de Perfil Corrompido
